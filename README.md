# Evil Portal HTML

This repository contains static **HTML, CSS, and JavaScript** templates created for **educational and learning purposes only**.

The project focuses on:
- Practicing **HTML, CSS, and JavaScript**
- Understanding the structure of **captive portals**
- Learning about **social engineering from a defensive perspective**
- Simulating user interfaces for **cybersecurity awareness and testing**

## Contents
- Multiple folders with different portal examples
- Simple and customizable HTML/CSS/JS code
- Resources designed for local testing and study
